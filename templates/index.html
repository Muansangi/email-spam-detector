How to get an App Password

  1. Go to Google Account > Security.
  2. Ensure 2-Step Verification is turned ON.
  3. Search for "App passwords" in the settings search bar.
  4. Create a new password for "Mail" and a Custom Name (e.g., "Sentry").
  5. Copy the 16-letter code and paste it here!

Note: Your normal email password will not work here due to Google's strict security policies.
